# LibNet

This is a book social network.

## Install command

To install:

1. Clone repo

2. `cd LibNet`

3. `pip3 install -r requirements.txt`

4. `cd sn`

5. Change path to db in file app.py

6. `python3 db.py` or `py db.py`

## Run command

To run use next command: `python3 app.py` or `py app.py`
